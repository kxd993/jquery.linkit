# LINK IT JQUERY PLUGIN #
**Link it** is a simple jquery plugin to attach link to elements

##Version##
0.2.0

##Usage##
  $('span').linkIt({
                    href : 'test.com',
                    target :'_blank' ,
                    text : 'deptrai'
                });
##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
